# graphql-node-seed
A GraphQL Node Server Seed based on TypeScript and graphql-yoga
## Get Started
- Clone this repo
`git clone https://github.com/jboothe/graphql-node-seed.git`
- cd into the project
`cd graphql-node-seed`
- npm install
- npm start
- Open http://localhost:4000
- Run the sample query
```
{
  beerMe
}
```
